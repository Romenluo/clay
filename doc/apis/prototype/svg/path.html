path标签

<path>用于定义一个路径，一个有下面d属性可选项：

  • 1.M：移动到（例子：'M10,20'）；
  • 2.L：画直线到（例子：'L10,20 11,21 12,22'）；
  • 3.H：画水平直线到（例子：'H10'）；
  • 4.V：画垂直直线到（例子：'V20'）；
  • 5.C：三次贝塞尔曲线；
  • 6.S：简化的贝塞尔曲线；
  • 7.Q：二次贝塞尔曲线Q；
  • 8.T：Q命令的简写命令；
  • 9.A：画圆；
  • 10.Z：闭合路径，不需要参数。

A 画圆 

<path d="Arx,ry x-axis-rotation large-arc-flag sweep-flag x,y"/>
  • 1.rx和ry分别表示弧的半长轴长度和半短轴长度
  • 2.x-axis-rotation是此段弧所在的x轴与水平方向的夹角，即x轴的逆时针旋转角度，负数代表顺时针旋转角度
  • 3.large-arc-flag为1表示大角度弧线，0表示小角度弧线
  • 4.sweep-flag为1表示从起点到终点弧线绕中心顺时针方向，0表示逆时针方向
  • 5.终点左边(x, y)
